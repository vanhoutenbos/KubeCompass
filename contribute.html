Contribution Guidelines

Comparison Page Template

Elke comparison page moet bevatten:

  1. Hero Section: Breadcrumb, title, reading time, last updated, helpful %
  2. Quick Decision: 3 opties met bullets en CTA buttons
  3. Tool Cards: 3 cards met badges, trust signals (GitHub stars, usage, testing), features, trade-offs, use cases
  4. Feature Matrix: Color-coded table ( green, orange, red) met 10 features
  5. Decision Wizard: 4 vragen met radio buttons, scoring algorithm, confidence calculation
  6. Implementation: Links naar docs, manifests, runbooks
  7. Feedback: Thumbs up/down buttons
  8. Related Comparisons: 4 cards linking naar andere pages

Decision Wizard Scoring

Elke wizard moet:

  • 4 vragen hebben (niet meer, niet minder)
  • Weighted scoring gebruiken (5 punten = strong match, 3 = good, 1 = neutral)
  • Confidence score berekenen (high: diff > 4, medium: diff > 2, low: diff 2)
  • Reasons array bijhouden per tool (waarom deze score?)

Getting Started

  1. Fork de repository: https://github.com/vanhoutenbos/KubeCompass
  2. Clone je fork: git clone https://github.com/YOUR-USERNAME/KubeCompass
  3. Create een branch: git checkout -b feature/my-comparison
  4. Maak je changes (volg de template)
  5. Test lokaal (open HTML files in browser)
  6. Commit: git commit -m "Add X vs Y comparison"
  7. Push: git push origin feature/my-comparison
  8. Open een Pull Request op GitHub

Principes

  • Production-Tested: Only recommend wat je zelf hebt getest of van trusted sources komt
  • Vendor Neutral: Geen vendor bias, eerlijke trade-offs
