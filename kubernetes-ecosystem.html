Interactieve Functionaliteit

Deze infographic kan interactief gemaakt worden door JavaScript toe te voegen die luistert naar klikken op de verschillende domeinen. Elk domein heeft een uniek ID in het SVG bestand.

Hoe maak je het interactief?

  1. Open het SVG bestand in een teksteditor
  2. Elk domein heeft een class="domain" en data-domain attribuut
  3. Voeg event listeners toe met JavaScript
  4. Link naar de juiste documentatiepagina's
  5. Voeg tooltips en hover effecten toe voor betere UX

Voorbeeld code:

document.querySelectorAll('.domain').forEach(domain => {
    domain.addEventListener('click', () => {
        const domainName = domain.dataset.domain;
        window.location.href = `docs/${domainName}.md`;
    });
});
